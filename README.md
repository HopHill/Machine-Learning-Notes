# 机器学习学习笔记

主要介绍机器学习中常用算法的基础知识，每个算法的介绍的通用架构为：

    算法的发展进程

    算法的基本原理或基本思想

    算法的基本流程

    算法的优缺点

    算法的改进措施

    算法在sklearn库中的模块

    算法在现实世界中的应用
